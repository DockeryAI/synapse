| **Schema Type** | **Use Case** | **SEO Impact** | **Implementation** |
|----------------|------------|---------------|-------------------|

| **LocalBusiness** | Local SEO | Map pack | NAP + hours |
| **Video** | Video content | Video carousel | Duration, thumbnail |
| **Recipe** | Food content | Recipe cards | Ingredients, time, nutrition |

**Implementation Code Example:**
```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "How to Write Converting Content",
  "step": [{
    "@type": "HowToStep",
    "text": "Research your audience"
  }]
}
```

#### Featured Snippet Optimization

**Snippet Types & Strategies:**

1. **Paragraph Snippets (63% of all snippets)**
   - 40-60 word answers
   - Start with "is" definition
   - Format: Question as H2, answer immediately below

2. **List Snippets (19%)**
   - Use numbered or bulleted lists
   - 5-8 items optimal
   - Clear, concise item descriptions

3. **Table Snippets (16%)**
   - Use proper HTML tables
   - Include headers
   - 3-5 columns maximum

4. **Video Snippets (2%)**
   - YouTube videos with timestamps
   - Clear titles and descriptions
   - Transcript inclusion

**Snippet Optimization Framework:**
```
[Question Keyword] + "is" + [Definition 40-60 words]
Then:
- Supporting point 1
- Supporting point 2
- Supporting point 3
```
