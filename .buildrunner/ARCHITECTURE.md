## Key Patterns

### API Pattern
All external API calls MUST go through edge functions in `supabase/functions/`.
Frontend components never call external APIs directly.

### State Management Pattern
- Server state: React Query / SWR for caching
- UI state: React Context or Zustand
- Form state: React Hook Form

### Security Pattern
- RLS (Row Level Security) on ALL database tables
- Auth tokens validated on every request
- Secrets stored in environment variables only

## Constraints

1. **Security:** RLS required, edge functions for external APIs
2. **UI Libraries:** Use ~/Projects/ui-libraries/ only (shadcn, aceternity, magic-ui)
3. **No Direct API Calls:** Frontend cannot call external APIs directly
