# Alpha Test Setup Checklist

**Complete these tasks before recruiting users:**

## 1. Technical Setup (30 minutes)

- [ ] Deploy latest build to staging/production URL
- [ ] Verify all features work on deployed version
- [ ] Test on mobile (iPhone + Android)
- [ ] Create 5 test accounts in Supabase
- [ ] Set up analytics tracking (optional)
- [ ] Test AI chat widget works (check ANTHROPIC_API_KEY)
- [ ] Test voice input works (check OPENAI_API_KEY for Whisper)
- [ ] Verify all API keys active (no rate limits)

## 2. User Recruitment (60 minutes)

- [ ] Identify 5 SMB owners (see target mix in ALPHA_TESTING_GUIDE.md)
- [ ] Send invitation email (template below)
- [ ] Schedule 2-hour testing sessions (Thu-Fri)
- [ ] Confirm availability and send calendar invites
- [ ] Send test credentials and pre-test instructions

## 3. Feedback Collection Setup (30 minutes)

- [ ] Create Google Form for post-test survey
- [ ] Set up Loom account for screen recordings
- [ ] Prepare Zoom links for one-on-one interviews
- [ ] Create feedback analysis spreadsheet
- [ ] Set up Slack/Discord channel for real-time feedback

## 4. Documentation (30 minutes)

- [ ] Review ALPHA_TESTING_GUIDE.md
- [ ] Prepare test scenario cards (printable PDFs)
- [ ] Create "Getting Started" quick guide for users
- [ ] Prepare list of questions for interviews

### Post-Test Survey (Google Form Questions)

**Section 1: Overall Experience**
1. Overall satisfaction (1-5 stars)
2. How likely are you to recommend Synapse? (0-10, NPS)
3. Would you use Synapse for your business? (Yes / Maybe / No)

**Section 2: Onboarding**
4. How easy was onboarding? (Very Easy / Easy / Neutral / Hard / Very Hard)
5. Did the AI understand your business? (Yes / Mostly / Somewhat / No)
6. Was the UVP wizard helpful? (Yes / No / Didn't use)
7. What was confusing about onboarding? (Open text)

**Section 3: Campaign Creation**
8. How easy was campaign creation? (Very Easy / Easy / Neutral / Hard / Very Hard)
9. Were the AI suggestions helpful? (Yes / No / Somewhat)
10. Were the generated posts relevant? (Yes / Mostly / Somewhat / No)
11. Could you easily edit posts? (Yes / No / Didn't try)
12. What was confusing about campaign creation? (Open text)

**Section 4: Features**
13. Which features did you use? (Check all: UGC contests, Hashtag builder, Email capture, Seasonal calendar, AI chat, Voice input, Mobile preview, Benchmark dashboard)
14. Most valuable features? (Select up to 3 from above list)
15. Most confusing features? (Select up to 3 from above list)

**Section 5: AI Chat**
16. Did you use the AI chat? (Yes / No)
17. If yes, was it helpful? (Yes / Somewhat / No)
18. Did you try voice input? (Yes / No)
19. If yes, did it work well? (Yes / No / Had issues)

**Section 6: Mobile**
20. Did you test on mobile? (Yes / No)
21. If yes, was mobile experience good? (Yes / No / Had issues)
22. What didn't work on mobile? (Open text)

**Section 7: Benchmarks**
23. Did you view the benchmark dashboard? (Yes / No)
24. Did benchmarks help you understand "what good looks like"? (Yes / Somewhat / No)
25. Were Day 3 pivots clear? (Yes / Somewhat / No / Didn't see)

**Section 8: Open Feedback**
26. What did you LOVE? (Open text)
27. What was CONFUSING? (Open text)
28. What's MISSING? (Open text)
29. Any other feedback? (Open text)

## Beta Launch Prep

**After alpha testing complete:**

- [ ] Analyze all feedback (see ALPHA_TESTING_GUIDE.md)
- [ ] Fix critical issues
- [ ] Re-test fixed issues
- [ ] Create beta user list (20-30 SMBs)
- [ ] Record demo video (Loom, 5-10 minutes)
- [ ] Write beta onboarding email
- [ ] Set up weekly feedback sessions
- [ ] Prepare beta launch announcement

**Beta User List (Target 20-30):**
- 8-10 local businesses
- 6-8 e-commerce
- 4-6 B2B/professional services
- 2-4 service businesses
- 2-4 content creators

**Beta Success Metrics:**
- 70%+ campaign completion rate
- 60%+ use video content
- 4+ out of 5 satisfaction score
- < 3 critical bugs reported
- Positive feedback on AI chat
